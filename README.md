# 実装したもの

- Todo の追加
- Todo の完了
- コメント追加
- localStorage でリロードしても Todo を残す

# やりたかった

# 処理

## Todo の追加

- Todo の入力
- +ボタンのクリック
- 入力値チェック
- localStorage への追加
- 画面表示の更新

## Todo の完了

- 完了ボタン(チェック)のクリック
- localStorage から Todo を削除
- 画面表示の更新
